# Entity-Salience-Aggregator

A production-ready Retrieval-Augmented Generation (RAG) pipeline built to query and extract insights from my research paper, "The Synthesis Gap in Regional News". The application combines a local-first embedding architecture with a high-performance cloud LLM to deliver fast, factually accurate answers. The model is uploaded within the phasefy.py file.
